# My personal nix/nixos/home-manager config

This config uses:
- den
- flake-file
- flake-parts
- home-manager
- import-tree

Refer to the [den documentation](https://vic.github.io/den)

For now the structure is:

```
├─modules
| ├─aspects (General aspects)
| ├─hosts (Where all hosts will eventually be defined, hardware and software capabilities)
| └─users (User configs and their specific aspects)
└─nixos (Plain autogenerated nixos configs I still need to integrate)
```

To run just:
- Update den input.
```console
nix flake update den
```
- Build

```console
# default action is build
nix run .#igloo

# pass any other nh action
nix run .#igloo -- switch
```
